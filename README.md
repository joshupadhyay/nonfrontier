# NonFrontier

Covering local models and the libraries used to run and train them! (HuggingFace's Transformers, Ollama)

Serves as a great refresh of Python and `uv` for me.

UV: Python toolchain, sets up virtual environment, package manager, all-in-one.

Don't forget to select the correct python interpreter in VSCode for import resolution!

You'll need a GROQ_API_KEY and an account at Modal to use `groq.py` and `modal_gpt_oss.py`. Note that you run the modal file with `modal run src/modal_gpt_oss.py`.
